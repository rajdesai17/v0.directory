## Why v0.directory

v0 is powerful, but the quality of output depends heavily on the prompt.

great prompts are often scattered across tweets, screenshots, or private notes.  
v0.directory exists to collect the ones that actually work and make them easy to find.

this is a community-driven library of practical v0 prompts, focused on real UI use cases.

## Submitting a Prompt

v0.directory uses **GitHub Issues** as its submission and moderation system.

When you submit a prompt:
1. a GitHub Issue is automatically created  
2. the prompt is reviewed by maintainers  
3. approved prompts are published to the directory  

If you have a prompt that consistently gives you good results, feel free to submit it.

## How It Works
