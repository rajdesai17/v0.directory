# v0.directory

A community-driven prompt directory for [v0](https://v0.dev) by Vercel.

![v0.directory Screenshot](./public/og-image.png)

## Overview

v0.directory is a curated collection of prompts, MCPs (Model Context Protocol servers), and custom instructions optimized for v0's AI coding assistant.

**Live Demo:** [LINK_PLACEHOLDER]

## Features

- **30+ Curated Prompts** - Across 8 categories including dashboards, landing pages, e-commerce, and more
- **MCP Directory** - Setup guides for popular MCP servers
- **Custom Instructions** - Pre-built instruction sets for v0
- **One-Click Copy** - Instantly copy any prompt to your clipboard
- **Search & Filter** - Find prompts by category or keyword
- **Community Submissions** - Submit your own prompts via GitHub Issues
- **Dark Mode** - Beautiful dark theme throughout

## Categories

| Category | Description |
|----------|-------------|
| Dashboards | Analytics, admin panels, data visualization |

| Animations | Micro-interactions, scroll effects |
| Apps | Full applications, chat, productivity tools |

## Tech Stack

- [Next.js 15](https://nextjs.org/) - React framework with App Router
- [Tailwind CSS v4](https://tailwindcss.com/) - Utility-first CSS
- [shadcn/ui](https://ui.shadcn.com/) - UI components
- [Framer Motion](https://www.framer.com/motion/) - Animations
- [TypeScript](https://www.typescriptlang.org/) - Type safety

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended) or npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/[USERNAME]/v0-directory.git
cd v0-directory
```

2. Install dependencies:
```bash
pnpm install
```

3. Set up environment variables:
```bash
cp .env.example .env.local
```

Add your GitHub token for the submission feature:
```
GITHUB_TOKEN=your_github_token
```

4. Run the development server:
```bash
pnpm dev
```

5. Open [http://localhost:3000](http://localhost:3000)

## Vercel Community Forum Blog Post

### Title

Introducing v0.directory - A Community Prompt Library for v0
```

### Post Content
```
Hey Vercel Community!

I'm excited to share a project I've been working on as a v0 Ambassador: **v0.directory** - a community-driven prompt library for v0.

## The Problem

If you've used v0, you know that the quality of your output depends heavily on the quality of your prompt. A detailed, well-structured prompt gets you production-ready code. A vague prompt gets you... something that needs a lot of iteration.

Writing great prompts takes time and practice. I wanted to make it easier for everyone to get amazing results from v0.

## The Solution

v0.directory is a curated collection of:

- **30+ Prompts** across 8 categories (dashboards, landing pages, e-commerce, animations, and more)
- **MCP Directory** with setup guides for Context7, Supabase, Stripe, and others
- **Custom Instructions** optimized for v0

Every prompt follows v0 best practices with clear sections, specific requirements, and detailed specifications.

## How It Works

1. Browse or search for a prompt
2. Click to view the full prompt
3. Copy with one click
4. Paste into v0 and watch the magic happen

## Built With v0

The entire site was built using v0 itself, showcasing what's possible:

- Next.js 15 with App Router
- Tailwind CSS v4
- shadcn/ui components
- Framer Motion animations
- TypeScript throughout

## Community Contributions

This is meant to be a community resource. You can:

- Submit prompts, MCPs, or instructions via the site
- Contributions are reviewed and credited
- Fork the project and self-host

I'm planning to add:

Would love to hear your feedback and see your prompt submissions!
